Blocksmurfer

The Blocksmurfer explorer is created by Coineva using Flask and Python Bitcoinlib. It can be used to search the {{ network_name }} blockchain for blocks, transactions, addresses and utxo's. You can easily run it from a local server using the source code from Github.

Currently the {% for _, nw in available_networks %} {{ nw|capitalize }}{% if loop.index == loop.length - 1 %} and{% elif not loop.last %}, {% endif %} {% endfor %} networks are supported.

Providers

Just like Bitcoinlib it retrieves blockchain data from various online service providers or a local bitcoin or Bcoin node.

For this website the following providers are used for the {{ network_name }} network: {% for provider in providers %} {{ provider[0] -}} {% if loop.index == loop.length - 1 %} and {% elif not loop.last %}, {% endif %} {% endfor -%}. To avoid too many request to external service providers the local providers bcoin and bitcoind are used whenever possible. Also transaction, block and address data are cached in a local SQLite or PostgreSQL database.

Bitcoinlib

Python Bitcoinlib is an open source bitcoin and other cryptocurrency library. You can use it to create and manage wallets, keys, transactions and interact with the blockchain through various service providers.
