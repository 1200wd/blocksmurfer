Blocksmurfer

The Blocksmurfer explorer is created by Coineva using Flask and Python Bitcoinlib. It can be used to search the blockchain for blocks, transactions, addresses and utxo's. You can easily run it from a local server using the source code from Github.

Just like Bitcoinlib it can retrieve data from various online service providers or work with a local bitcoin or Bcoin node.

Python Bitcoinlib is an open source bitcoin and other cryptocurrency library. You can use it to create and manage wallets, keys, transactions and interact with the blockchain through various service providers.
